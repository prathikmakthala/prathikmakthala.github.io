Sky Sail Insights: Passenger Sentiment Analysis

Unveiling passenger sentiments on British Airways using NLP and Machine Learning.

Project Overview

This project utilizes Natural Language Processing (NLP) to convert complex, unstructured customer feedback from British Airways passengers into clear, actionable insights. By systematically decoding and classifying sentiments, the goal was to identify key themes and patterns that influence overall satisfaction and loyalty.

The analysis provides data-driven recommendations to British Airways for targeted service improvements, demonstrating a deep understanding of how passenger experiences shape a brand's reputation.

Key Features & Analysis Pipeline

  • **Dataset:** Analysis was performed on a Kaggle dataset of 2,000 British Airways customer reviews.
  • **Data Preprocessing:** Implemented a full preprocessing pipeline including tokenization, lemmatization, and stop-word removal.
  • **Sentiment Modeling:** Utilized TextBlob for baseline sentiment extraction and trained multiple classification models (SVM, Naive Bayes, XGBoost) on TF-IDF vectors, achieving 90% accuracy in predicting sentiment.
  • **Geographical Insights:** Applied unsupervised topic modeling (LDA) to discover latent topics and sentiments specific to passengers from different countries, such as the US, UK, and Australia.

Technologies Used

  • NLTK
