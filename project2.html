High-Performance Payments Backend

At Razorpay, I built and maintained high-performance backend APIs using Python (Flask) to support critical payment flows, including UPI, IMPS, and NEFT. The system was engineered for high availability and scalability, successfully handling over one million daily payment requests while maintaining 99.99% uptime and minimal transaction latency.

Key Responsibilities & Achievements

  • Architected and developed REST APIs for the end-to-end payment lifecycle.
  • Refactored monolithic payment modules into containerized microservices using Docker and Kubernetes, improving deployment efficiency and scalability.
  • Dramatically improved performance by optimizing PostgreSQL queries and implementing Redis caching, which lowered average API response times from 300ms to 200ms during peak traffic.
  • Designed and implemented event-driven processing pipelines with AWS SQS to ensure reliable transaction settlement.

Technologies Used

  • Python
  • Flask
  • Docker
  • Kubernetes
  • PostgreSQL
  • Redis
  • AWS (SQS)
