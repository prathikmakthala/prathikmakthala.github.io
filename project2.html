High-Performance Payments Backend

A scalable backend supporting over one million daily payment requests.

Project Overview

At Razorpay, I built high-performance backend APIs using Python (Flask) to support UPI, IMPS, and NEFT payment flows. The system was designed for high availability, enabling over one million daily payment requests with 99.99% uptime and minimal transaction latency.

Key achievements included refactoring payment modules into containerized microservices using Docker and Kubernetes, and optimizing PostgreSQL queries and Redis caching to lower average API response times from 300ms to 200ms during payment peaks.

Technologies Used

Python Flask PostgreSQL Redis Docker Kubernetes AWS
