LLM-Powered Medical Chatbot

An AI-driven application for medical inquiries.

Project Overview

This project is an AI-driven application featuring an LLM-powered medical chatbot. The goal was to provide reliable and context-aware responses to user queries in the medical domain, focusing on system reliability and performance.

The application was built using a combination of natural language processing techniques and modern web frameworks to create an intuitive and responsive user interface.

Technologies Used

Python Flask React.js NLP Google's Flan-T5 Hugging Face Transformers
