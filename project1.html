LLM Fine-Tuning for a Medical Chatbot

A full-stack AI application with a React-based frontend and a fine-tuned T5 language model.

Project Overview

This project is a full-stack, AI-driven application featuring an intelligent medical chatbot. It consists of two main components: a sophisticated machine learning pipeline for model fine-tuning, and a modern, responsive frontend for user interaction.

The core of the application is a `google/flan-t5-base` language model, which was fine-tuned on a specialized question-answering dataset to become an expert in medical inquiries. The user-facing interface is a clean and intuitive chatbot, built with **React**, allowing users to interact with the powerful fine-tuned model in real-time.

Key Features & Pipeline Stages

  • **Frontend:** A responsive and user-friendly chatbot interface built with React.
  • **Data Processing:** A robust pipeline using Pandas and NLTK for cleaning, preprocessing, and lemmatizing text data.
  • **Model Fine-Tuning:** The T5 model was fine-tuned using the Hugging Face `transformers` and `datasets` libraries on a PyTorch backend.
  • **Evaluation:** The model's performance and accuracy were measured using the ROUGE metric.
  • **Full-Stack Integration:** The project demonstrates the end-to-end process of training a specialized language model and creating an interface to serve it to users.
